Kuniko Bot – Privacy Policy

Last Updated: 2025-01-01

1. Data Collected

Kuniko Bot does not collect, store, or retain personal information. The bot only processes user input temporarily when executing commands.

2. Temporary Processing

The following data may be accessed temporarily by Discord’s API during command execution:

  • User ID (required internally by Discord)
  • Text entered in slash commands

This information is not stored or logged.

3. Data Storage

Kuniko Bot does not save:

  • Messages
  • Usernames
  • User IDs
  • Server information
  • Conversation logs

4. Data Sharing

No data is shared with third parties. No analytics or tracking systems are used.

5. User Rights

Since no data is stored, users do not need to request data deletion or correction.

6. Changes

The developer may update this Privacy Policy at any time. Continued use of the bot implies acceptance of any updates.

7. Contact

For questions or issues, contact the bot owner via Discord.
